<!--more -->

实现网页黑白办法其实很简单,只需要设置 filter 滤镜为 grayscale 就可以了。

``` css
    html{
         filter: grayscale(100%); 
        -webkit-filter: grayscale(100%); 
    }
```
常用几个值
grayscale ：将图像转换为灰度图像。
blur：可以实现高斯模糊效果。
Saturate ：设置饱和度。
Invert : 颜色反转。
Contrast  ：设置对比度。
